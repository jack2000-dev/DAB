# Act

Put insights to work. Implement. Monitor. Iterate.

## What "Act" looks like

- Implement a new business strategy
- Make changes to a product, website, or process
- Stop doing something the data showed isn't working
- Continue investing in something the data shows works

## How to act

1. **Discuss with leadership** — decide how to implement and take actions based on findings
2. **Communicate the decision** — to all affected stakeholders
3. **Monitor** — compare datasets over time (monthly, quarterly, annually) to see if changes are working
4. **Iterate** — re-enter the process from Ask if needed

## Monitoring

- Set up dashboards (Tableau, Metabase, Looker)
- Define success metrics before acting (so you can measure)
- Set review cadence: monthly, quarterly, yearly
- A/B test where possible

## Five Whys

When acting on a finding, ask "why" five times to surface the root cause:

1. Why is conversion dropping? → Checkout takes too long.
2. Why? → 5-step flow.
3. Why? → Each field validates server-side.
4. Why? → Legacy form library.
5. Why? → Never replaced after migration.

→ Replace the form library, not the marketing copy.

- What is your final conclusion?
- How could the team and business apply these insights?
- What next steps would stakeholders take?
- Is there additional data you could use to expand findings?
